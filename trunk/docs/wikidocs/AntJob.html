A JobDetail that Executes a Native Command such as an ANT Build Script


Creating a JobDetail to invoke an ANT Build Script
JobDetail nativeANTJob = new JobDetail("AntJob", "AntGroup", org.quartz.jobs.NativeJob.class);
JobDataMap jobDataMap = new JobDataMap();
jobDataMap.put(org.quartz.jobs.NativeJob.PROP_COMMAND, "ant -f C:\\dev\\build.xml mytarget -Dmy.property=Hello");
jobDataMap.put(org.quartz.jobs.NativeJob.PROP_CONSUME_STREAMS, "true"); // Optional: Captures ant output
nativeANTJob.setJobDataMap(jobDataMap);
